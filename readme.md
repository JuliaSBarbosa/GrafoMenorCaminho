# 📌 Projeto: Grafos e Algoritmos de Menor Caminho

## 📖 Descrição

Este projeto tem como objetivo estudar e aplicar conceitos de **grafos** e **algoritmos de menor caminho**, com foco em soluções computacionais para problemas reais.

O projeto faz parte da disciplina de **Integração e Entrega Contínua**, do curso de **Desenvolvimento de Software Multiplataforma**, da **FATEC Itapira**.

---

## 🎯 Objetivo

Desenvolver uma solução computacional baseada em algoritmos de menor caminho, fundamentada por uma Revisão Sistemática da Literatura.
